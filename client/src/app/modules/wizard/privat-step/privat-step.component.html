Add your PrivatBank card

According to the instructions indicated on the PrivatBank API website, you need to:

  • Log in to the Privat24 account for physical. persons using the link
  • Enter your username and static password
  • Enter the dynamic OTP password received in SMS on your mobile phone
  • Go to the menu section "All services" -> "Business" -> "Merchant"
  • Bind the card to work with the merchant and specify it in the field below
  • Indicate the IP address of our server from which requests will be made: 127.0.0.1
  • Check the services necessary for work (others can be disabled):
    • Statement for merchandise merchants
    • Statement for merchandise merchants
    • The balance of the merchandise merchandise
    • Cartographic information
  • Click "Next"
  • Confirm OTP Password
  • Registration is over. Merchant is assigned an ID and Password. The merchant is in the “Test” status, which will be completely enough.
  • Indicate the ID and Password (key) of the Merchant, and indicate the number of the selected card in the fields below.
